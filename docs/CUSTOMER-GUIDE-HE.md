## אפשרות 1: הזמנה באתר

### שלב אחר שלב:

**1. בחרו תאריך וכמות אורחים**
- בחרו תאריך מהלוח
- קבעו כמה אורחים (1-20) עם כפתורי +/-

**2. בחרו שעה**
- תראו את כל השעות הפנויות לתאריך שבחרתם
- כל שעה מראה כמה שולחנות נשארו
- לחצו על השעה שמתאימה לכם

**3. העדפות** (לא חובה)
- **ישיבה**: בפנים, בחוץ, או בבר
- **עישון**: אזור ללא עישון או אזור מעשנים
- **אלרגיות**: אגוזים, חלבי, גלוטן, פירות ים, ביצים, סויה

**4. פרטים אישיים**
- שם
- טלפון (05X-XXXXXXX)
- בקשות מיוחדות (יום הולדת, כסא לתינוק, שולחן ספציפי)

**5. אישור**
- בדקו את הסיכום ואשרו
- מוכן! תקבלו הודעת אישור.

> כל התהליך לוקח פחות מדקה.

## אפשרות 2: הזמנה בצ'אט (טלגרם / וואטסאפ)

אפשר להזמין שולחן בשיחה רגילה עם הבוט שלנו — בעברית, אנגלית או ערבית.

### דוגמאות:

**הזמנה פשוטה:**
```
אתם: היי, אני רוצה שולחן ל-4 הערב
בוט: בדקתי — יש מקום ב-19:00, 20:00 ו-21:30. מה מתאים?
אתם: 20:00
בוט: מה השם?
אתם: דני כהן
בוט: מספר טלפון?
אתם: 0501234567
בוט: מצוין! שולחן ל-4 הערב ב-20:00 על שם דני כהן. נתראה!
```

**כשאין מקום:**
```
אתם: שולחן ל-6 ביום שישי
בוט: מצטער, אין שולחן פנוי ל-6 ביום שישי. רוצים להיכנס לרשימת המתנה?
אתם: כן
בוט: הוספתי אתכם. אם יתפנה מקום תקבלו הודעה עם 15 דקות לאשר.
```

**ביטול:**
```
אתם: צריך לבטל את ההזמנה להערב
בוט: ביטלתי. מקווים לראות אותכם בפעם הבאה!
```

**שאלות:**
```
אתם: מתי אתם פתוחים ביום שישי?
בוט: ביום שישי אנחנו פתוחים 11:00-16:00. רוצים להזמין?

אתם: מה הכתובת?
בוט: אחוזה 130, מרכז אש, רעננה.
```

### טיפים:
- **דברו בטבעיות** — הבוט מבין עברית, אנגלית וערבית
- **תהיו ספציפיים** — "שולחן ל-4 מחר ב-20:00" יותר מהיר מ-"אני רוצה להזמין"
- **אפשר לבטל** — פשוט תגידו ותבוטל
- **הבוט מכיר אתכם** — לקוחות חוזרים מזוהים לפי הטלפון, ההעדפות שלכם שמורות
