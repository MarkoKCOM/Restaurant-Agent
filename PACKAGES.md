# Sable — Packages & Pricing

## Launch Offer (First 5 Restaurants)
**₪299 one-time** — Full reservation system. No monthly fee. Limited to 5 spots.
Includes: reservations, widget, dashboard, table management, operating hours.

|---------|----------|
| Web booking widget (embeddable) | Yes |
| Owner dashboard (Today, Reservations, Guests, Settings) | Yes |
| Auto table assignment | Yes |
| Operating hours + special dates | Yes |
| Reservation reminders | Yes |
| Waitlist with auto-match | Yes |

| Occupancy heatmap | Yes |
